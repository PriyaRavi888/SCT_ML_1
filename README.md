# SCT_ML_1
Task 01 - House Price Prediction using Linear Regression (SkillCraft Internship)


## Project Summary
A simple linear regression model to predict house prices using three features:
- `square_feet`
- `bedrooms`
- `bathrooms`

Built for the SkillCraft Technology internship task. This single Jupyter notebook demonstrates dataset creation, model training, prediction, and visualization.

## Files
- `SCT_ML_1.ipynb` — Jupyter Notebook with the full project (data, model, plots).
- `README.md` — Project description.
